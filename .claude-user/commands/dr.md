# Design Review (Quick Alias)

Abbreviated command for `/design-review`. Launch ux-ui-designer for UI/design review.

## Usage

```bash
# Review current branch changes
/dr

# Review specific PR
/dr 234

# Review preview URL
/dr https://preview-abc123.app
```

## Notes

This is a shortcut for the full `/design-review` command. See `design-review.md` for complete documentation.
