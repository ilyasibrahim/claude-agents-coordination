# Review (Quick Alias)

Abbreviated command for `/review`. Launch code-quality agent for code review.

## Usage

```bash
# Review recent changes
/r

# Review specific file
/r src/models/model.py

# Review directory
/r src/pipeline/
```

## Notes

This is a shortcut for the full `/review` command. See `review.md` for complete documentation.
