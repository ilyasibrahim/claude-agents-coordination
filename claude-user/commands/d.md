# Deploy (Quick Alias)

Abbreviated command for `/deploy`. Deploy application to specified environment.

## Usage

```bash
# Deploy to production
/d

# Deploy to staging
/d staging

# Deploy to development
/d dev
```

## Notes

This is a shortcut for the full `/deploy` command. See `deploy.md` for complete documentation.
